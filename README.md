## Tech Stack

**Frontend:** CSS , Bootstrap

**Backend:** Node, Express 

**Database:** Mongodb


## Run Locally

Clone the project

```bash
  git clone https://github.com/nagarajRPoojari/yelpcamp
```

Go to the project directory

```bash
  cd yelpcamp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
```bash
  node app.js
```
